# VestaDV
Vesta ruimtelijk energiemodel voor de gebouwde omgeving

Het ruimtelijk energiemodel Vesta berekent het energiegebruik en de CO2-uitstoot van de gebouwde omgeving (onder andere woningen, kantoren, winkels en ziekenhuizen) en de glastuinbouw voor de periode van 2010 tot 2050. Daarnaast kan het model de effecten berekenen van gebouwmaatregelen en gebiedsmaatregelen voor warmtelevering in termen van vermeden CO2-uitstoot, energiegebruik, investeringskosten en financiÃ«le opbrengsten.
